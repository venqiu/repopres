# Netizen's Guide to Preserving Github Repositories

This is an introductory guide for archivists, creative technologists, and netizens to understand the different options for preserving a github repository. This guide was written by Koel Chu, Ven Qiu, and Haoran Wang. 

## Live site

Main site (GitHub Pages):

- `https://venqiu.github.io/repopres/`

If you are viewing a fork, the URL will be something like:

- `https://<username>.github.io/repopres/`
